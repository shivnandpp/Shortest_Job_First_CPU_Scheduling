# Shortest_Job_First_CPU_Scheduling
This project implements the Shortest Job First (SJF) CPU Scheduling Algorithm (Non-Preemptive) using the C programming language.The program simulates how an operating system schedules processes based on their burst time, selecting the process with the smallest burst time among the processes that have already arrived.It calculates and displays the Waiting Time (WT) and Turnaround Time (TAT) for each process along with the average waiting and turnaround times.
